# Create Minimal

Utiliser ce chemin pour :
- creer un skill simple
- poser un scaffold personnel propre
- preparer une base legere sans architecture lourde

## Objectif
Generer directement le socle personnel :
- `SKILL.md`
- `governance/`
- `references/`
- `scripts/`
- `assets/`
- `agents/openai.yaml`

## Regles
- creer les dossiers canoniques meme si leur contenu reste leger
- ne pas creer de module de gouvernance decoratif
- ne pas creer de schema decoratif
- ne pas embarquer de data verite externe
- utiliser l'ordre fixe de sections dans `SKILL.md`

## Sortie attendue
- scaffold complet
- chemin de lecture court
- validation rapide possible sans audit lourd
